# 命相卜

> 命、相、卜二十六门传统术数与民俗象征工具: **固定算法排盘 × 按法源边界审慎解读**。

一个面向传统文化研习与个人反思的术数应用: 所有排盘算法(天文历法、干支节气、农历置闰、安星诀、遁甲布局、地占衍卦……)都在浏览器本地用 TypeScript 实现; AI 文字解读与「相」类多模态识图可配置主流大模型 API 或 OpenAI 兼容接口。中式模块保留「玄机子」道门解读形象; 西占、吠陀、Ifá、Vastu、玛雅、阿兹特克、Weton、人类图等异域或现代体系走中性解读员口径, 不冒称国学正统。

## 快速开始

```bash
npm install
npm run dev        # 仅绑定 127.0.0.1:5174
```

想让 AI 开口解读: 右上角「AI 设置」中选择模型服务商, 填入 API Key、接口地址和模型名。内置 OpenAI、Anthropic、Google Gemini、DeepSeek、通义千问、Kimi/Moonshot 与 OpenAI 兼容接口模板。密钥只保留在当前页面内存中, 不写入仓库、本地项目配置或浏览器持久化存储; 刷新页面后需重新输入。

## 三大分类 · 二十六门

**命 (12)** — 生辰/姓名代入固定公式: 八字四柱 · 紫微斗数 · 韩国四柱사주 · 西方占星 · 吠陀占星 · 藏历占星 · 数字命理 · 姓名学五格 · 人类图 · 玛雅卓尔金历 · 阿兹特克圣历 · 爪哇 Weton

**相 (4)** — 观察物理对象, 提取可见特征并映射到象义分类: 面相(三停五官) · 手相(含 Hast Rekha) · 风水(玄空飞星+八宅+命卦) · Vastu Shastra
> 流程: 所选多模态模型按固定观察协议输出结构化特征分类 → 本地映射为传统断语 → AI 按当前模块法源综合解读

**卜 (10)** — 当下起一课, 答具体之问: 易经六爻(京房纳甲全装卦) · 梅花易数(体用) · 塔罗 78 牌 · 卢恩符文 · 奇门遁甲(时家转盘) · 大六壬(九宗门四课三传) · 太乙神数(岁计式) · 伊法 Ifá(256奥都) · 沙卜 Ilm al-Raml · 马达加斯加 Sikidy

## 传统对齐说明 (给民俗/玄学工作者)

可核验的核心规则尽量按各自传统口径实现, 并用古籍课例、立成表和历法锚点复核关键规则; 未覆盖处会在盘面标明边界:

- **八字/韩国四柱**: 子时换日三派可选(子初换日默认/夜子时分派/子正民用); 旺衰按 得令(月支本气旺相休囚死评分, 《渊海》人元司令按交节实足天数另列供参、不入评分)/得地(本气中气余气通根, 逢冲减力)/得势(三干党众) 综合, 从格专旺仅作提示; 另据《穷通宝鉴》十天干十二月表列调候用神(寒暖燥湿), 与扶抑并列合参; 真太阳时可开启, 未开启时盘面提示。韩国四柱同样要求出生地 UTC 偏移, 并提供出生地经度/真太阳时校正, 默认经度为首尔
- **六爻**: 京房纳甲排出八宫归宫→六亲→伏神(飞神标注)→月建日辰旺衰/月破/日辰冲爻分暗动(冲旺相静爻)日破(冲休囚静爻)→六冲六合卦→变爻六亲→进退神/爻卦反伏吟/用神十二宫长生/应期候选(旬空出空、月破填合、入墓冲开), 23 点换日; 用神只做关键词候选, 不冒称完整人工断卦
- **大六壬**: 四课三传九宗门取传——贼克/知一/涉害(依《六壬大全》: 涉归本家历数所经支之本气与寄宫干, 起点本家不计)/见机察微缀瑕/遥克/八专/别责/昴星/伏吟(迤逦刑传+自刑颠倒+子卯取冲)/返吟(无依·井栏射), 并以《大全》14 则古籍课例复核; 贵人用古传歌诀(甲戊庚牛羊…壬癸蛇兔藏), 卯至申为昼。另按问事取十二类神(财物青龙妻财/官禄贵人官鬼/疾病白虎/失物玄武/婚姻六合天后/口舌朱雀勾陈等)并列六亲天将; 增《毕法赋》高置信结构断语区(旬空、三传递生、六害入传、三刑逢冲、闭口旬首旬尾互加、贵人顺逆/昼夜贵加临), 另据盘面结构拟撰两贵受克、进退茹连珠、间传隔位等断语并明标"拟断·非赋文原句"以别于赋文原句, 绝嗣课等胎产细法标待考; 年命上神与全量课体格局仍需师承判断
- **奇门**: 当前实际算法为时家奇门·转盘·拆补法, 拆补三元(仲上孟中季下), 增旬空/驿马/门迫门制/六仪击刑/十干入墓/十干克应格局(81组三奇六仪, 依《统宗》)/八门九星旺相休囚/宫制门迫/日时干落宫用神; 天盘伏/反吟与值使门伏/反吟分立标注。置闰法、飞盘法仅作审计选项提示, 不按该法排盘
- **紫微**: 《全书》七级庙旺利陷表(14主星+昌曲辅弼羊陀火铃)、长生十二神、小限(原文口诀)、斗君、红鸾天喜天姚天刑天哭天虚孤辰寡宿, 另补天官天福天厨天空龙池凤阁蜚廉破碎华盖咸池天才天寿解神天巫天月阴煞截路空亡旬空(依《全书》《全集》流传安星诀录表归纳、非逐字古籍口诀, 截路/空亡合称截空之说标待考)、指定太岁年流年四化/流曜/岁前将前十二神、可选流月/流日/流时(斗君起流正月顺行, 流月干叠四化); 闰月三流派可选(折半法默认); 四化派别未做选项, 当前采用通行南派表, 庚壬干他派异说在盘面/aiContext 标注
- **玄空风水**: 二十四山下卦飞星(三元龙阴阳顺逆)、三元九运、旺山旺向/上山下水/双星判局(八运子山午向=双星到向等名局锚点)、东西四命卦配宅; 兼向替卦(《沈氏玄空学》替星诀逐山)、城门诀、流年飞星(4.5°兼线自动切替卦)
- **藏历**: 浦派(Phugpa)精确历算(Janson 公式, 洛萨锚点 2000-2027 全对), 生年以洛萨为界(可与农历属相不同), 重日缺日闰月俱准; 帕卡依传统按虚岁性别递推(藏历本无"生年定命卦"之法, 不套用汉地公式)
- **太乙**: 岁计式依《太乙金镜式经》——太乙九宫(乾一离二…与洛书异)、十六神、文昌(天目)、计神(起寅逆行)、始击、主客算与大小参将(弃十法), 以《武经总要》立成表(甲子局/1984 丙子局)验证; 可填日期按立春或冬至岁首口径定岁计年。月计式已接积月推法, 但**积月基数为类比推定、非文献确证正法(待考)**; 阴局为阳局公式的顺逆对称实验接线(推定/待考——太乙岁计以积年单向推步、传统本无阴阳遁换向, 二至分遁属日计/时计与奇门); 日计/时计基数未见可靠原文, 仅审计提示

## 技术要点

- **天文引擎** `src/core/astro.ts`: Meeus 太阳/月亮黄经(月亮对照书中算例误差 0.001°)、Standish 行星根数、上升/天顶、均时差、Lahiri 会差
- **中式历法** `src/core/chinese.ts`: 定朔定气农历(无中气置闰)、二十四节气(分钟级)、干支、八字(子时三派)、十二长生、人元司令、刑冲合害、旺衰引擎、大运、神煞
- **藏历引擎** `src/core/phugpa.ts`: Phugpa 真月/重缺日/洛萨精确算术(整数运算, Janson arXiv:1401.6285)
- **仪式交互**: 掷铜钱(3D 翻转)、扇形抽牌、符文石、沙盘点划、奥佩雷链——真随机来自你的手
- **玄机子真身** `src/components/XuanJiZi.tsx` + `src/core/master.ts`: 中式模块使用的道门仙人 SVG 形象——眼球实时追鼠标(rAF)、眨眼呼吸胡须飘动; 12 种表情 × 9 种姿势 × 8 件可由 AI 指令触发的法器(罗盘/拂尘/铜钱/签筒/折扇/卷轴/茶盏/龟甲); 起盘、掷钱、揭盘、报错、输入密钥等流程有对应反馈。中式 AI 解读时模型可通过文中 `[[指令]]`(如 `[[掐指]]` `[[取罗盘]]`)触发界面动作, 指令流式滤除不上屏; 异域和现代模块使用中性解读员提示词, 不输出舞台/法器指令。
- **AI**: 文字解盘与图像观测支持主流大模型 API 和 OpenAI 兼容接口; 前端统一走 `src/core/ai.ts`。API Key 只保留在当前页面内存中, 不写入仓库或本地项目配置。
